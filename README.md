My Portfolio Website
A simple and responsive **Personal Portfolio Website** built using **HTML5** and **CSS3**. This project is designed to present my profile in a clean and professional way.

## Pages

* **Home** – Introduction and basic information about me.
* **Contact** – A page where visitors can find my contact details.

## Technologies Used

* HTML5
* CSS3

## Features

* Clean and responsive layout
* Easy navigation
* Beginner-friendly project
* Lightweight and fast loading

## Purpose

This project was created to practice front-end web development using HTML and CSS while building a simple portfolio website.
